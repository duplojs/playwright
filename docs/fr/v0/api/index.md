# Référence API

`@duplojs/playwright` s'organise autour d'un `Website`, de pages, de composants et d'helpers pour écrire des tests Playwright plus lisibles.

## [🌐 Website](/fr/v0/api/website)
Point d'entrée principal. Porte la `Page`, le `BrowserContext` et les helpers globaux.

## [🧩 Component](/fr/v0/api/component)
Déclare un composant réutilisable à partir de son élément principal et de ses éléments nommés.

## [🤝 Component Interaction](/fr/v0/api/componentInteraction)
Construit des interactions personnalisées réutilisables pour les composants.

## [📄 Page](/fr/v0/api/page)
Déclare une page avec son chemin, son élément principal et ses composants.

## [💥 Actions](/fr/v0/api/actions)
Liste les actions prêtes à l'emploi à appliquer sur les éléments d'un composant.

## [✅ Assertions](/fr/v0/api/assertions)
Liste les assertions prêtes à l'emploi à appliquer sur les éléments d'un composant.
